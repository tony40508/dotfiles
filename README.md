# dotfiles

```
ln -sf ~/dotfiles/nvim/init.vim ~/.config/nvim/init.vim
# others also
```
## plugins
### tmux
```
prefix(Ctrl-a) + :
source-file ~/.tmux.conf
```
- tmux-resurrect
```
prefix + Ctrl-s - save
prefix + Ctrl-r - restore
```
## shortcut
```
; -> select opened file by Ctrl-j/k

Ctrl-o + d -> delete selected file

space + f -> search file

m -> insert/delete file in nerd tree
```
