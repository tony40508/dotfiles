# dotfiles

```
ln -sf ~/dotfiles/nvim/init.vim ~/.config/nvim/init.vim
# others also
```
## plugins
### coc.nvim
```
After update coc-settings.json
Restart coc server by command :CocRestart
```

### tmux
```
prefix(Ctrl-a) + :
source-file ~/.tmux.conf
```
- tmux-resurrect
```
prefix + Ctrl-s - save
prefix + Ctrl-r - restore
```
## shortcut
```
space + f -> search file

m -> insert/delete file in nerd tree
```
